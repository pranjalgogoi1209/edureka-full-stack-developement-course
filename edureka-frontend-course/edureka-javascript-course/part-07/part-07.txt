# Loops => We use these concepts to run a particular code multiple times.

* for loop => It is used to perform/run a code specific number of times and we kind of specify that number.
    syntax => 
        for(initialization of variable; condition; increment/decrement of variable){
            ...lines of code
        }
* while loop => It used to perform a task multiple times as long as the condition holds true.
    syntax =>
        while(condition){
            ...lines of code
        }

* do while loop =>
    syntax => 
        do{
            ...lines of code
        }while(condition)
